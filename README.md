# YouthConnect

## Description
Ce projet consiste à développer une application sociale et interactive pour les jeunes permettant de découvrir, créer et partager des activités ou événements.

Les utilisateurs peuvent publier des activités, rejoindre celles des autres et interagir afin de créer une communauté autour de leurs centres d’intérêt.

## Acteurs

### Acteur principal
Utilisateur
- créer un compte
- publier une activité
- rejoindre une activité
- consulter les activités

### Acteur secondaire
Administrateur
- gérer les utilisateurs
- modérer les contenus

## Technologies
Backend : Spring Boot / Jakarta EE  
Frontend : HTML, CSS, JavaScript  
Base de données : MySQL
